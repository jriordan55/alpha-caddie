Select Devig Books

Choose which sportsbooks count toward the CS (consensus) column and how strongly each one weighs in. For head-to-head markets you can also pick a method that changes how tight the “fair” price sits versus the posted lines. Apply & rescan updates the +EV table.

Picking a method

Methods only affect two-way matchups (not win/top-N outrights). They change how the app turns posted odds into a single fair price for the CS column—more or less conservative.

Tip: Start with Multiplicative or Average. Use Worst case when you want the stingiest fair price for your side. If unsure, compare a couple of methods and see which CS lines up with how you read the market.

+EV Bets

Scan matchup and outright bets in one place, sorted by Model EV so the biggest model edges float to the top. Narrow the list with the filters, then compare model prices to book prices column by column.

Weather

Use the same weather controls as Model O/U and Hole Hangout. Set temp, wind, humidity, and Condition to match the expected day; that updates fair win chances on every row here. Choosing Default for condition resets the numbers to the baseline weather snapshot (then adjust sliders again if needed).

Pricing

Mode and Skill match the rest of the app: switch them when the goal is to weight recent form, this course, or a specific part of the game. Model % and Model EV update to reflect that choice.

Devig settings

Open Devig to choose which books feed CS, split weights between them, and (for head-to-heads) pick a method that nudges consensus tighter or looser versus the board. Win and place markets use the selected books without that matchup-only control.

Tip: Try Average or Multiplicative first. Use Worst case when you want a cautious read on an edge.

Model O/U

Pick a Market (round score, props, and so on), then read each golfer’s model edge at the listed lines. Use Round to match Thursday through Sunday. Switch Prob / Price to read edges as percentages or as fair American odds. Set Line and the over and under odds to mirror a ticket; that column highlights when the line matches.

Table and chart

Tap column headers to sort. Green and red cells flag where the model disagrees with the book’s implied price. Choose one golfer in Golfer to open the bar chart and scan over probability across lines.

Weather

Dial in temp, wind, humidity, and Condition to match the forecast. Default condition resets the sliders to the baseline snapshot; then nudge again for a custom day. These controls update fair numbers together with Pricing.

Pricing

How to use it
Default mode locks Skill to Default as well (no pillar tilt). Recent form favors players trending up or down. Course history favors players with a strong record at this event. Skill focus lets you choose a spelled-out pillar (Total SG, Off the tee, Approach, Around the green, Putting, Tee to green). The same Mode and Skill row appears anywhere Pricing is shown.

Historical Trends

Compare past rounds to the model line for one golfer and stat. The header shows who and which event. Filters hide rounds that do not match chosen weather bands, the Course dropdown (when not All), or optional Current course only (this week’s venue from projections).

How to use the sidebar

Golfer and market
Select who to study and which stat the bars represent.
Rounds and line
Set how many recent rounds to include and which Line to use for over or under hit rates and the small EV readout.
Hit rates and top table
See over and under frequency in the window, then scroll the league table for the same line.

Round Matchups

Browse head-to-head and three-ball markets. Each row lists both players, a model win percentage, estimated EV, and the best posted price on the board.

Market and round

Use Market to switch tournament, single-round, or three-ball lists. Set Round on the Model O/U tab first; this table follows that choice. Adjust Weather and Pricing on other tabs to update fair win chances here.

Pricing

The Pricing row mirrors the rest of the app: pick a mode when the read should lean on form, this course, or a skill pillar.

Outrights

Choose a market (win, places, cut, and so on), then compare the Model column to each book. Tap headers to sort. Use Refresh projections after updating odds outside the app so the table matches the latest file.

Weather and pricing

Weather lives on Model O/U, +EV, or Hole Hangout but still moves fair chances here. Use the Pricing row to align with form, course, or skill the same way as on other tabs.

Hole Hangout

The course under the tabs applies to every screen. Pick Hole and Player (field follows the round on Model O/U). Results refresh when you change weather, pricing, hole, player, or live scenario; Run simulation forces an immediate refresh. The same settings reproduce the same sample shot path.

Live inputs

Turn on Live inputs to set lie, yards to the hole, which shot it is, and putt length. Use it to mimic standing in a specific spot instead of a generic tee shot.

Weather

These sliders stay in sync with the rest of the app; change them the same way as on Model O/U to reflect wind and conditions.

Pricing

Pricing mode shifts how bullish the sim is on birdies versus mistakes, using the same Mode and Skill as everywhere else.
